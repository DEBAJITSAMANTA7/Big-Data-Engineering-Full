A DataFrame is a two-dimensional labeled data structure (like a table).

Each column is a Series.  
You can mix data types across columns.

Syntax:
    pd.DataFrame(data)
python
Copy
Edit
df = pd.DataFrame([[10, 20, 30],
                   [40, 50, 60],
                   [70, 80, 90]])
print(df)
print(df.shape, df.index, "\n", df.values)
